Redmine Show Issue Assignee Avatar Plugin
=========================================

This is Redmine plugin to show issue assignee avatar on issue page.
(Usually avatar on issue page is issue auther avatar.)

Requirements
============

* Redmine 2.6.x or higher.

Installation
============

In Redmine folder,
```
$ cd plugins
$ git clone https://github.com/two-pack/redmine_show_issue_assignee_avatar.git
```
Finally restart Redmine.
